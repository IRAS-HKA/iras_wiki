## Masterprojekt Ausschreibung
1. Erstellen mit Template [RoboTrainer Teams](https://hskarlsruhede.sharepoint.com/:w:/t/RoboTrainer/ERY-t0MXKWBEg4ZSezCBspABdvGdAnL_NDWCRRD5rMcs3g?e=Dlkpft)
	- Projekt Code und Dateiname müssen übereinstimmen
	- Format: YYws/ss_2LetterLastName_ShortProjectTitle
	- Example: 24ws_HE_RoboTrainer_Ganganalyse
2. An Robert Weiß per Mail schicken robert.weiss@h-ka.de
> 	Hallo Herr Weiß,
> 	im Anhang finden Sie eine neue Ausschreibung für ein Forschungs und Entwicklungsprojekt für den Masterstudiengang RKIM.
> 	Bitte laden Sie es an entsprechender Stelle in ILIAS hoch.
3. In ILIAS in diesen Ordner hochladen [ILIAS](https://ilias.h-ka.de/goto.php?target=cat_870471&client_id=HSKA).
4. Studis finden und zusagen
	- Studis müssen auf Ausschreibung unterschreiben
	- Betreuer muss unterschreiben
5. Studis laden die unterschriebene Ausschreibung selber in ILIAS hoch und melden sich zum entsprechenden Projekt Code an.
